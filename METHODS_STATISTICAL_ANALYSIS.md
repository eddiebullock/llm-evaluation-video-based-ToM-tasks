# Statistical Analysis (Methods subsection)

Model performance was quantified as classification accuracy over valid trials. Per-mental-state accuracy was computed for each label; *Neutral* was excluded from per-state summaries. For EU-Emotions, mean accuracy on high-intensity labels was compared with mean accuracy on paired low-intensity variants (labels containing “low intensity”). Basic emotions were defined as happiness, sadness, fear, anger, disgust, and surprise; on EU-Emotions this comprised the six base labels and their low-intensity variants, with all other non-neutral states classified as complex. On Mindreading, basic emotions were defined by exact match to the six canonical English terms only (no synonym expansion).

Analyses were conducted in Python 3 using NumPy, Pandas, SciPy, and statsmodels via `analysis/run_study_analysis.py`. One-sided binomial tests assessed whether each model exceeded chance (25%, 4-AFC). Accuracy CIs used the Wilson score method (95%).

Model versus human performance on EU-Emotions used two-sided two-proportion *z*-tests against validation benchmarks. Video-only models were compared to O’Reilly et al.’s (2016) facial-expression validation accuracy (*M* = 63.00%, *n* = 1,231). O’Reilly et al. also report body-gesture (77%) and social-scene (72%) benchmarks; facial expression was used as the closest match to full-face video clips, although no single human modality perfectly matches the models’ video presentation. Audio-only Gemini 3 Flash was compared to Lassalle et al.’s (2019) UK vocal-expression benchmark (*M* = 45.19%, *n* = 427). No human benchmark was applied to multimodal EU-Emotions or Mindreading. Original human validation used 6-AFC; comparability with the present 4-AFC design is limited. Effect sizes are reported as Cohen’s *h*.

Pairwise model comparisons within each dataset and input condition used Fisher’s exact tests (six comparisons among four models), with Bonferroni correction (family-wise α = .005). Mindreading video-only cross-model analyses used the 581-trial intersection with valid video input. Gemini 3 Flash modality ablations were compared with two-proportion *z*-tests and Bonferroni correction across six modality contrasts (α = .005). Cross-dataset generalisation (EU-Emotions vs. Mindreading, video-only) used two-proportion *z*-tests with Bonferroni correction across four models (α = .01). Post-hoc power for human-benchmark comparisons used `statsmodels.stats.power.NormalIndPower` (Cohen’s *h*, α = .01, two-sided).
